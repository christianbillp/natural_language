# Twitter
To use TwitterMiner you need to set up an developer account for Twitter and get the following information:
consumer_key, consumer_secret, access_token and access_secret.

Then create a file called twitter_conf.txt with the keys on one line in this pattern: "consumer_key,consumer_secret,access_token,access_secret"
